## CLI 

The file `rosalind.py` contains a cli that can be used to run and test solutions. 

### Testing
```bash
$ python rosalind.py NAME_OF_PROBLEM --test
```
### Running
```bash
$ python rosalind.py NAME_OF_PROBLEM --dataset PATH_TO_DATASET
```
